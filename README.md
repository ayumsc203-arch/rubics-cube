
# 🧩 Solve Rubik's Cube with Ayu

An interactive web project to visualize and solve the Rubik's Cube.  
Built with **HTML, CSS, and JavaScript**, this project demonstrates cube rotations, solving algorithms, and a playful UI.

---

## 🚀 Features
- Visual representation of a Rubik's Cube
- Interactive cube rotations
- Step-by-step solving animations
- Beginner-friendly explanations

---

## 📂 Project Structure
- `index.html` → Main entry point
- `style.css` → Cube styling and layout
- `script.js` → Cube logic and interactions

---

## ⚡ Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/ayumsc203-arch/rubics-cube.git
   ```
2. Open `index.html` in your browser.
3. Explore and enjoy solving the cube!
